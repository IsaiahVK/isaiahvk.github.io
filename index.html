Low Cost Carrier Flight Opportunities¶

Intro¶

Recently, Spirit Airlines has been struggling to stay afloat, filing for bankruptcy twice in the past year. Ultra-low-cost-carriers (ULCCs) like Spirit operate on a point to point network, offering nonstop routes that the legacy airlines (Delta, American, and United) are unable or unwilling to compete with.

Goal¶

For my project, I plan to analyze flight bookings and economic conditions to determine which routes could potentially save struggling low cost carriers like Spirit.

Data¶

The Bureau of Transportation Statistics (BTS) has a robust dataset of commerical airlines itineraries, including pricing, origin and destination, stops, and more. This data is free to access and includes a 10% sample of the data for that quarter.

Methodology for Importing the Data¶

The BTS has publicly available zip files with csv files inside. The first time we use data from a specific year and quarter, we must download it. After this initial download, data is stored to a parquet file in Google Drive for quicker loading.

In [3]:
# @title Downloading the Data

def download_and_save_bts(year, quarter):
    # We have two tables of use from BTS
    tables = {
        "Market": f"https://transtats.bts.gov/PREZIP/Origin_and_Destination_Survey_DB1BMarket_{year}_{quarter}.zip",
        "Coupon": f"https://transtats.bts.gov/PREZIP/Origin_and_Destination_Survey_DB1BCoupon_{year}_{quarter}.zip"
    }

    for name, url in tables.items():
        filename = os.path.join(project_path, f"db1b_{name.lower()}_{year}q{quarter}.parquet")

        # Check if we already have the parquet file to skip the download
        if os.path.exists(filename):
            print(f"File {filename} already exists. Skipping download.")
            continue

        print(f"Downloading {name} data for {year} Q{quarter}...")
        response = requests.get(url)

        if response.status_code == 200:
            with zipfile.ZipFile(io.BytesIO(response.content)) as z:
                csv_file = [f for f in z.namelist() if f.endswith('.csv')][0]
                with z.open(csv_file) as f:

                    df = pd.read_csv(f, low_memory=False)

                    # Save as Parquet
                    df.to_parquet(filename, index=False)
                    print(f"Saved {name} to {filename}")
        else:
            print(f"Failed to download {name}.")

# Run it for 2025 Q1
download_and_save_bts(2025, 1)

Preparing the Main Dataframe¶

Once data is downloaded and imported, it must be merged onto a single table. This is done by joining on the itinerary ID with an inner join. Before joining, the columns we need are selected to avoid issues with using too much memory.

These are the important variables that we will be using:

Variable Meaning

| ItinID | Itinerary ID | MktFare | Fare paid for each ticket | Origin | Airport code the itinerary originates at | Dest | Airport code the itinerary ends at | OpCarrier| Operating carrier of the itinerary |Passengers| Amount of passengers on the reservation |MktMilesFlown| The amount of miles flown for the whole itinerary

Why are nonstop flights important?¶

Nonstop flights, if adequate demand is met, are better for all parties. Putting passengers on a single plane traveling less overall miles is more cost-effective for the airline, if the demand exists. It is also much more convenient from a customer perspective. Nonstop flights are shorter, and there is no worry about missing a connection.

Misleading Insight¶

From this graph, it may seem that connecting flights are more expensive. This isn't necessarily true, because legacy carriers operate far more connecting flights, and the cost of a ticket on a legacy carrier is more expensive. Airlines use dynamic pricing models, so the fare is not necessarily reflective of the cost to run that flight. Lets take a look at what happens if we break down these stats by legacy vs. low cost carriers.

# Define the carrier groups
legacy_carriers = ['AA', 'DL', 'UA']
lcc_carriers = ['NK', 'WN', 'B6', 'F9', 'G4', 'AS'] # Spirit, Southwest, JetBlue, Frontier, Allegiant, Alaska

def get_carrier_type(carrier):
    if carrier in legacy_carriers:
        return 'Legacy'
    elif carrier in lcc_carriers:
        return 'Low Cost'
    else:
        return 'Other'

# Apply the categorization
df_combined['CarrierType'] = df_combined['OpCarrier'].apply(get_carrier_type)

# Filter for the main comparison (excluding 'Other')
df_filtered = df_combined[df_combined['CarrierType'] != 'Other']

# 1. Calculate the average fare grouped by Break and Carrier Type
# Resulting structure: Rows = Break (0, 1), Columns = Carrier Type (Legacy, Low Cost)
fare_stats = df_filtered.groupby(['Break', 'CarrierType'])['MktFare'].mean().unstack()

# 2. Setup Plotting Variables
labels = ['Nonstop', 'Connecting']
legacy_means = fare_stats['Legacy']
lcc_means = fare_stats['Low Cost']

x = np.arange(len(labels))  # label locations
width = 0.35  # width of the bars

fig, ax = plt.subplots(figsize=(10, 6))

rects2 = ax.bar(x + width/2, lcc_means, width, label='Low Cost (NK, WN, etc.)', color='#ffcc00') # Yellow

# 4. Add labels, title and axes ticks

def autolabel(rects):
    for rect in rects:
        height = rect.get_height()
        ax.annotate(f'${height:.2f}',
                    xy=(rect.get_x() + rect.get_width() / 2, height),
                    xytext=(0, 3),  # 3 points vertical offset

                    ha='center', va='bottom', fontweight='bold')
